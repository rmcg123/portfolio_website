The Guindex Python Package

In August 2023 I created and released my first python package. The package is called guindex and can be installed via `pip install guindex`. It's a small package which talks to the API of the website guindex.ie (which some friends and I run) and returns pandas DataFrame of selected data from the guindex project. The guindex is a collaborative data collection project examining the pricing of pints of Guinness around Ireland.

The page for the package on the Python Packaging Index (PyPI) is here including some basic examples. The code for the package is available on my GitHub here. A couple of mini projects using data obtained through this package are showcased in this portfolio:

  • Using pint price data to examine the geographic variation of pint pricing in Dublin postal areas in 2017-18 here and
  • Creating interactive maps showing the pub locations and most recent submitted data by county here.
