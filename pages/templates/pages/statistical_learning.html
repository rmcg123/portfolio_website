An Introduction to Statistical Learning Solutions

In late 2021 I started to do the applied exercises from the book "An Introduction to Statistical Learning" (ISLR) by Gareth James, Daniela Witten, Trevor Hastie and Robert Tibshirani. The book is available at https://www.statlearning.com/. I found this book to be excellent free resource to learn statistical learning techniques and would highly recommend it to anyone. I did these exercises using R and wrote up the solutions using R Markdown. I got as far as chapter 8 before I was rudely interrupted by securing paid employment.

The solutions for the applied exercises from chapters 1 through 8 can be viewed in a markdown file on my GitHub here
